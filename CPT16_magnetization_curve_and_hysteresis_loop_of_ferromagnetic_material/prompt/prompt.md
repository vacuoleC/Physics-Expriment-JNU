# 铁磁材料磁化曲线与磁滞回线实验数据分析脚本生成提示文档

## 1. 实验概述
本实验使用示波器法测绘铁磁材料的基本磁化曲线和磁滞回线，通过测量励磁电压和感应电压，计算磁场强度H、磁感应强度B和磁导率μ，并确定饱和磁感应强度Bs、剩磁Br和矫顽力Hc等关键磁学参数。

## 2. 输入数据格式
### 2.1 基本磁化曲线数据
- 输入为CSV格式或Python列表，包含三列数据：
  - `U`：励磁电压(V)
  - `U1`：励磁回路取样电阻电压(V)
  - `U2`：积分电容两端电压(V)
- 示例数据：
  ```
  U, U1, U2
  0.5, 105.6e-3, 46.77e-3
  1.0, 214.0e-3, 79.58e-3
  1.2, 284.0e-3, 93.47e-3
  1.5, 454.4e-3, 116.9e-3
  1.8, 645.1e-3, 133.2e-3
  2.0, 813.1e-3, 141.9e-3
  2.2, 994.6e-3, 149.8e-3
  2.5, 1.251, 157.5e-3
  2.8, 1.499, 164.0e-3
  3.0, 1.650, 167.0e-3
  ```

### 2.2 饱和磁滞回线数据
- 输入为Python字典，包含以下关键点的电压值：
  - 正向饱和点：`U1S`, `U2S`
  - 正向剩磁点：`U1R`, `U2R`
  - 正向矫顽力点：`U1D`, `U2D`
  - 反向饱和点：`U1S_prime`, `U2S_prime`
  - 反向剩磁点：`U1R_prime`, `U2R_prime`
  - 反向矫顽力点：`U1D_prime`, `U2D_prime`
- 示例数据：
  ```python
  hysteresis_data = {
      'U1S': 1.699, 'U2S': 169.0e-3,
      'U1R': 9.880e-3, 'U2R': 36.80e-3,
      'U1D': 60.84e-3, 'U2D': 0.0,
      'U1S_prime': 1.660, 'U2S_prime': 167.0e-3,
      'U1R_prime': 0.0, 'U2R_prime': 37.60e-3,
      'U1D_prime': 65.0e-3, 'U2D_prime': 0.0
  }
  ```

## 3. 实验常数与计算公式
### 3.1 实验常数（固定值）
```python
N = 30          # 励磁线圈匝数
L = 0.060       # 样品平均磁路长度(m)
R1 = 2.0        # 励磁回路取样电阻(Ω)
n = 150         # 测量线圈匝数
S = 80e-6       # 样品截面积(m²)
C2 = 20e-6      # 积分电容(F)
R2 = 10e3       # 积分电阻(Ω)
```

### 3.2 核心计算公式
1. **磁场强度H**：
   $$H = \frac{N}{L} \cdot \frac{U_1}{R_1} \quad (\text{单位：A/m})$$

2. **磁感应强度B**：
   $$B = \frac{C_2 R_2}{n S} \cdot U_2 \quad (\text{单位：T})$$

3. **磁导率μ**：
   $$\mu = \frac{B}{H} \quad (\text{单位：H/m})$$

## 4. 数据分析步骤
### 4.1 基本磁化曲线分析
1. 读取基本磁化曲线数据
2. 根据公式计算每一组数据对应的H、B、μ值
3. 绘制B-H基本磁化曲线
4. 绘制μ-H磁导率曲线
5. 确定饱和磁感应强度Bs（B-H曲线趋于平缓时的B值）

### 4.2 磁滞回线分析
1. 读取饱和磁滞回线关键点数据
2. 计算各关键点对应的H和B值
3. 绘制完整的饱和磁滞回线（关键点顺序：正向饱和点→正向剩磁点→正向矫顽力点→反向饱和点→反向剩磁点→反向矫顽力点→回到正向饱和点）
4. 从磁滞回线中提取关键参数：
   - 饱和磁感应强度Bs：正向饱和点的B值
   - 剩磁Br：H=0时的B值
   - 矫顽力Hc：B=0时的H值

## 5. 输出要求
### 5.1 数据表格输出
1. 基本磁化曲线计算结果表：包含U, U1, U2, H, B, μ
2. 磁滞回线关键参数表：包含Bs, Br, Hc

### 5.2 图表输出
1. B-H基本磁化曲线（带数据点和拟合线）
2. μ-H磁导率曲线（标注最大磁导率点）
3. 饱和磁滞回线（标注Bs, Br, Hc关键点）

### 5.3 图表格式要求
- 所有图表必须包含标题、坐标轴标签（含单位）和图例
- 坐标轴比例适当，数据点清晰可见
- 关键参数点用不同颜色或标记突出显示
- 图表风格统一，使用科学计数法表示数值
- 保存为高分辨率PNG格式（300dpi）

## 7. 注意事项
1. 计算磁导率μ时，注意避免H=0的情况（会导致除零错误）
2. 磁滞回线绘制时，确保曲线关于原点对称
3. 饱和磁感应强度Bs应取B-H曲线趋于饱和时的最大值
4. 剩磁Br是H=0时的B值，矫顽力Hc是B=0时的H值
5. 所有计算结果应保留适当的有效数字（通常3-4位）
6. 以示例数据为基础，将示例数据的config放到脚本开头，方便修改为新的实验数据
